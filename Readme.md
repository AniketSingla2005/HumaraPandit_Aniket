# AstroDesk CRM

## Overview

AstroDesk CRM is a simple web-based client management system designed for astrologers to maintain basic client records.

The application allows users to:

* Add new clients
* View all clients
* Search for existing clients
* Delete client records
* Track consultation status

Client information is stored using the browser's LocalStorage, allowing data to persist after page refresh.
